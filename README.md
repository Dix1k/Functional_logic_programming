Этот репозиторий создан для выполнения лабораторных работ по курсу функционального и логического программирования. Здесь представлены решения задач на языке Prolog, демонстрирующие различные аспекты логического программирования: от основ унификации до создания полноценных игровых приложений.

## 🗂️ Структура лабораторных работ

| Название | Технологии | Краткое описание |
|----------|------------|------------------|
| **[LR1](https://github.com/Dix1k/Functional_logic_programming/tree/main/Prolog/Prolog/vlasov/LR1)** | Prolog | **Унификация декларативной программы**: факты и правила, определение отношений, простейшие предикаты, генеалогическое дерево. |
| **[LR2](https://github.com/Dix1k/Functional_logic_programming/tree/main/Prolog/Prolog/vlasov/LR2)** | Prolog | **Механизм поиска решений**: бэктрекинг, внутренняя/внешняя цели, логические функции (И, ИЛИ, НЕ), база знаний предпочтений. |
| **[LR3](https://github.com/Dix1k/Functional_logic_programming/tree/main/Prolog/Prolog/vlasov/LR3)** | Prolog | **Рекурсия в Prolog**: поиск пути в лабиринте, вычисление суммы и степени, геометрическая прогрессия, генерация случайных чисел. |
| **[LR4](https://github.com/Dix1k/Functional_logic_programming/tree/main/Prolog/Prolog/vlasov/LR4)** | Prolog | **Внутренние базы данных**: assert/retract, save/consult, кэширование результатов (числа Фибоначчи), журнал группы с меню. |
| **[LR5](https://github.com/Dix1k/Functional_logic_programming/tree/main/Prolog/Prolog/vlasov/LR5)** | Prolog | **Хороший стиль и игра "Быки и коровы"**: слияние списков, НОД, реализация игры с оконным интерфейсом, магический квадрат. |
